# YOUR LEARNINGSPACE

A simple Blog Content Management System made to practice new technologies ;HTML, CSS, JavaScript, PHP, and MySQL — built with love by a beginner 

---

##  Features

###  Frontend:
-  Home page with list of blog posts (title, image, short excerpt)
-  Single post view page (full blog content)
-  About page
-  Contact page

###  Admin Panel (PHP Backend):
-  Admin login system
-  Create new blog posts (title, content, image)
-  Edit posts
-  Delete posts
-  Search or filter posts by title/category

---

## Technologies Used
- HTML + CSS for layout & styling
- JavaScript for interactivity (like menus)
- PHP for backend logic and form handling
- MySQL for storing blog posts
- Organized folder structure (css/, js/, admin/, images/, etc.)
